Study Buddy v0.1.0

Study Buddy is an AI-powered study assistant designed to help students learn through conversation.

This is the first public version of the project. Users can ask questions, get explanations, and interact with an AI assistant directly from the terminal.

Features
AI-powered question answering
Simple terminal-based interface
Fast setup and lightweight design
Easy to modify and expand
How to Run
Install dependencies:
pip install google-genai and add your Gemini API key.
Run the application:
python study_buddy.py
Version

Current Version: v0.1.0

Future updates may include chat history, improved prompts, study tools, and a graphical interface.
